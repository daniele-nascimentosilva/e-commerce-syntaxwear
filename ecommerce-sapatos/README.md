# Projeto E-commerce SyntaxWear

Este é o repositório para o projeto de e-commerce SyntaxWear, focado na venda de sapatos. O objetivo é criar uma loja online moderna e responsiva, utilizando as melhores práticas de desenvolvimento web.

## Estrutura do Projeto

A estrutura de pastas e arquivos foi organizada para facilitar o desenvolvimento e a manutenção do projeto:

```
ecommerce-sapatos/
├── assets/
├── css/
│   ├── base.css
│   ├── components.css
│   ├── footer.css
│   ├── header.css
│   ├── hero.css
│   ├── layout.css
│   ├── product-card.css
│   ├── product-grid.css
│   ├── reset.css
│   └── variables.css
├── fonts/
├── images/
└── index.html
```

### Descrição das Pastas e Arquivos:

*   **`index.html`**: O arquivo principal HTML que serve como a página inicial do e-commerce.
*   **`assets/`**: Destinado a armazenar recursos diversos que não se encaixam nas outras categorias (ex: arquivos JavaScript, dados JSON, etc.).
*   **`css/`**: Contém todos os arquivos de estilo CSS do projeto, organizados da seguinte forma:
    *   `reset.css`: Folha de estilo para resetar os estilos padrão dos navegadores, garantindo consistência.
    *   `variables.css`: Define variáveis CSS (cores, fontes, espaçamentos) para facilitar a manutenção e padronização.
    *   `base.css`: Estilos base para elementos HTML (body, h1, p, a, etc.).
    *   `layout.css`: Estilos relacionados ao layout geral da página (grid, flexbox, posicionamento de seções).
    *   `components.css`: Estilos para componentes reutilizáveis da interface (botões, cards, formulários).
    *   `header.css`: Estilos específicos para o cabeçalho do site.
    *   `hero.css`: Estilos para a seção de destaque (hero section) da página inicial.
    *   `product-card.css`: Estilos para os cartões de produtos individuais.
    *   `product-grid.css`: Estilos para a grade de exibição dos produtos.
    *   `footer.css`: Estilos específicos para o rodapé do site.
*   **`fonts/`**: Para armazenar arquivos de fontes personalizadas utilizadas no projeto.
*   **`images/`**: Para armazenar todas as imagens do site (logotipos, banners, fotos de produtos, ícones, etc.).

## Como Começar

Para visualizar o projeto localmente, siga os passos abaixo:

1.  **Clone o repositório** (se aplicável):
    ```bash
    git clone <URL_DO_REPOSITORIO>
    cd ecommerce-sapatos
    ```
2.  **Abra o `index.html`**: Simplesmente abra o arquivo `index.html` em seu navegador web preferido.

## Desenvolvimento

### Estilos CSS

Os estilos são organizados de forma modular para facilitar a localização e modificação. Certifique-se de seguir a convenção de nomenclatura e a estrutura existente ao adicionar novos estilos ou componentes.

### Favicon

O favicon está configurado para ser carregado a partir de `images/favicon.ico`. Certifique-se de colocar o arquivo do ícone nesta pasta.
